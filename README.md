# 🏭 Sistema de Gestión de Mantenimiento Industrial

Aplicación web para gestión integral de mantenimiento industrial basada en normas de inocuidad y calidad.

## 🚀 Características

- 📊 Dashboard con métricas en tiempo real
- 📦 Inventario de equipos con hojas de vida
- 📋 Editor de POEs (Procedimientos Operativos Estandarizados)
- 📅 Planificador de mantenimiento
- 📄 Generación de reportes profesionales en PDF
- ✅ Trazabilidad completa de operaciones
- ⚠️ Enfoque en normas de inocuidad alimentaria

## 🛠️ Tecnologías

- Python 3.9+
- Streamlit
- SQLite
- ReportLab (PDF)
- Pandas

## 📋 Requisitos de Inocuidad

- Advertencias de inocuidad obligatorias en POEs
- Clasificación por zonas (Blanca/Gris)
- Control de lubricantes grado alimenticio

## 👥 Uso

1. Configurar datos de la empresa
2. Registrar equipos
3. Crear POEs con procedimientos paso a paso
4. Programar mantenimientos
5. Generar reportes en PDF
